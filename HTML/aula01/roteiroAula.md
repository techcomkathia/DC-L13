# Aula 01: Introdução ao HTML

Esta é a primeira aula do curso de Desenvolvimento Web, dedicada à introdução aos conceitos básicos do HTML (HyperText Markup Language). Aqui, você aprenderá os fundamentos para criar suas primeiras páginas web.

## Repositório do Curso
Acesse o repositório completo do curso em: [https://github.com/usuario/DC-L13](https://github.com/usuario/DC-L13)

## Download do Visual Studio Code
Para acompanhar as aulas, recomendamos o uso do Visual Studio Code como editor de código. Baixe a versão mais recente em: [https://code.visualstudio.com/download](https://code.visualstudio.com/download)

## Assuntos Abordados
- **Instalação do editor de código**: Como instalar e configurar o Visual Studio Code.
- **Configuração e atalhos**: Personalização do editor e uso de atalhos para aumentar a produtividade.
- **Estrutura básica do HTML**: Entendendo a estrutura fundamental de um documento HTML.
- **O que são tags**: Conceito e função das tags HTML.
- **Atributos de uma tag**: Como adicionar atributos às tags para modificar seu comportamento.
- **Primeira página web**: Criando e visualizando sua primeira página HTML simples.
