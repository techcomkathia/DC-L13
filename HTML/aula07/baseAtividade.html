Prática de Fontes e Tipografia

Nesta atividade, você irá estilizar textos utilizando diferentes famílias de fontes, pesos, cores e alinhamentos. Leia atentamente cada texto e aplique os estilos no CSS.

Este texto deve utilizar uma fonte padrão, simples e de fácil leitura, semelhante às fontes mais comuns utilizadas em navegadores.

Este texto deve utilizar uma fonte com aparência mais estilizada, apenas para títulos ou destaques, não indicada para textos longos, pode utilizar serifas.

Peso das Fontes

Este parágrafo deve apresentar um texto visualmente mais fino e delicado, transmitindo leveza na leitura Utilize o valor de 300 ou light.

Este parágrafo deve apresentar um peso equilibrado, sendo confortável para leitura contínua utilize o valor de 400 ou normal.

Este parágrafo deve apresentar um texto mais espesso e marcante, ideal para chamadas de atenção ou destaques importantes Utilize o valor de 700 ou bold.

Este texto deve ficar centralizado, sendo utilizado geralmente em títulos ou mensagens de destaque.

Este texto deve ficar alinhado à direita, simulando um comportamento menos comum em textos longos.

Combinação de Estilos

Este parágrafo utiliza uma combinação básica: fonte padrão, peso equilibrado e alinhamento tradicional. Observe como a leitura se mantém confortável.

Este parágrafo combina uma fonte diferente, com maior peso visual e alinhamento centralizado, criando um efeito de destaque.

Este parágrafo combina uma fonte estilizada, peso mais leve, alinhamento diferenciado e uma cor de destaque. Ele serve como exemplo de uso pontual e não para textos longos.

Utilize este arquivo para experimentar combinações diferentes e entender como a tipografia influencia a experiência do usuário.
